Hashtags and Accessibility

When using hashtags, make sure they're formatted correctly. For example, writing #accessibilitymatters instead of #AccessibilityMatters can make it difficult for screen readers to interpret the text correctly.

Images and Alt-Text

Make sure all images have Alt text. Alt text should always be descriptive and end with a full stop.
